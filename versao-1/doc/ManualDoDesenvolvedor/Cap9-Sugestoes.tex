Com a finalidade de melhorar o projeto, sugere-se que os tópicos abaixo
sejam incorporados de modo que o código e formulação teórica sejam
ampliados a outros casos.

\item Espeficicação:

\item Apresenta-se a seguir os requisitos funcionais.

\item Formulação Teórica:
\begin{itemize}
\item Adicionar a IPR que abrange reservatórios estratificados, ou seja,
reservatórios com mais de uma camada e possibilitar o cálculo utilizando
os modelos empíricos.
\item Adicionar a IPR Futura, método utilizado para prever produtividade. 
\item Implementar outros regimes de escoamento, como o permanente e transiente.
\item Considerar os cálculos para poços horizontais ou direcionais.
\end{itemize}
\item Formulação do Código:
\begin{itemize}
\item Permitir que o usuário entre com os dados na forma de arquivo de disco.
\item Permitir que o usuário salve os resultados de pressão e vazão calculados
pelo método escolhido.
\end{itemize}
\end{itemize}
