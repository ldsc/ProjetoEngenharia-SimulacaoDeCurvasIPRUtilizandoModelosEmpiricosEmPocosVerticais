
\chapter{Teste\label{chap:Teste}}

\lhead{\thechapter - Teste} 

Todo projeto de engenharia passa por uma etapa de testes. Neste capítulo
apresentamos alguns testes do software desenvolvido. Estes testes
devem dar resposta aos diagramas de caso de uso inicialmente apresentados
(diagramas de caso de uso geral e específicos).

\section{Teste 1\label{sec:Teste-1}: Entrada de dados}

O software inicia com a entrada de dados referentes ao problema como:
tipo de fluido, raio de poço, viscosidade, fator volume formação do
fluido e etc. Os dados são inseridos via teclado pelo usuário \ref{fig:Software-Entrada-Dados}.
É importante notar que os tipos de dados inseridos está condicionado
ao método de IPR que será utilizado nos cálculos.
